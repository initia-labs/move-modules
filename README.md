# Initia Move Modules

A collection of Move modules for the Initia ecosystem — includes both **production utility modules** deployed on-chain and **example modules** used in documentation and tutorials.

## Modules

### Utility Modules

| Module                                 | Description                                                                   |
| -------------------------------------- | ----------------------------------------------------------------------------- |
| [dex-utils](./dex-utils)               | DEX utility library for route swap simulations with price impact calculations |
| [vip-utils](./vip-utils)               | VIP protocol helpers for vote weight conversion and operator vesting queries  |
| [nft-transfer](./nft-transfer)         | Cross-chain NFT transfer over IBC with automatic recovery                     |

### Example Modules

| Module                                 | Description                                                                   |
| -------------------------------------- | ----------------------------------------------------------------------------- |
| [todo](./todo)                         | Simple todo list app demonstrating Move fundamentals                          |
| [launch](./launch)                     | Token launch toolkit — coin wrapper, lock staking, and vesting schedules      |
| [swap-transfer](./swap-transfer)       | Atomic swap-then-transfer operations via IBC or OP Bridge                     |
| [minitswap-router](./minitswap-router) | Routing and simulation for Minitswap / Stableswap pools                       |
